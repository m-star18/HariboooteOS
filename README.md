## Prerequisites

Cコンパイラ: gcc 5.4.0 <br>
アセンブラ: Gas(GNU assembler) 2.26.1 <br>
リンカ: GNU ld 2.26.1 <br>
x86エミュレータ: QEMU2.5.0 <br>

## Usage

1. Clone this repository.
2. Run `make run` to start the HariboooteOS.(QEMU)
